Installation

First download the plugin file zip file, then there are two ways for installing:
 
1) The best way to install is via the Plugins > Add New Plugins section in the Wordpress Backend.
 
2) You can also manually bu unzipping plugin file in /wp-content/plugins directory
 
Once the plugin is installed please activate.

Configuration

Through the WooCommerce TrackingMore -> Settings you have to insert your TrackingMore API key and select which companies you will going to use to track the shippings. To generate a valid API Key just click the link you find under the API Key section.

Backend integration

Once you inserted a valid API Key and selected the companies you will use to track shippings, in each Order details page you will find an "TrackingMore" metabox through which you can insert the tracking info. Once a tracking has been created, it will show the current shipping status

Frontend integration

In each order details page, the WooCommerce TrackingMore plugin will show a timeline for each shipping

Shortcode

Through the special wcafw_order_status it is possible to track any order in any page. It will render a form in which the user can enter the order id. The form will then load the order shipping timelines
